Drone Dash

Description

Link to Live Demo: Click Here

Drone dash is a dashboard for surveillance using drones. Drone dash provides role based access, reports, instructions to the pilot, flight status, communication between the clients and the admin, livestreaming, uploading schedule and much more...

Drone dash is built using Angular, NodeJS, PostgreSQL and Hasura GraphQL Engine, with file uploads on Aws S3 Bucket.

Drone Dash Provides many features such as: SMS and Email Integration for notifications Report Validation by Admin. Role based access. (Roles being admin, semi admin, client and pilot) Messaging Service Flight stats and Logs and much more. Please check the live url for a demo.
